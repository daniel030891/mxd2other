EXPORTAR PROYECTOS MXD A OTROS FORMATOS

1. Herramienta que permite la obtención de todos los proyectos mxd que se ubiquen
   dentro de un directorio, ya sea dentro de otros directorios, etc.

2. Permite seleccionar el formato de salida:

   Formatos soportados:
    - PDF
    - PNG
    - JPEG
    - TIFF

3. Establece la resolución necesaria (por defecto esta resolución (dpi) es de 96)

4. Establece la calidad de la exportación (Solo disponible en formato PDF)
   Nivel de calidad:

    - BEST (Muy buena)
    - BETTER (Buena)
    - NORMAL (Normal)
    - FASTER (baja)
    - FASTED (muy baja)
